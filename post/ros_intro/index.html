A Quick Note

Hello! And welcome to my first post about my journey in F1TENTH. To begin this journey, I learned as much ROS as I could in order to solidy a solid framework for dealing with multiple sensors and data pipelines. Since ROS1 and ROS2 use the same core concepts, I decided to use ROS noetic as my distribution since I am new to ROS. ROS2 has more features and tools, which will only complicate learning the basics. ROS2 has better long term prospects, however transitioning from ROS to ROS2 is a small hurdle once ROS is known.
